Timirbaev Project - Courier Management System 1.0
Cross-platform Qt application for courier service management
Qt-приложение для управления курьерской службой с автоматической генерацией документации

Автообновляемая документация

https://kiruha1111.github.io/DZ_cross-platform_c/

Структура проекта

  • build/ - папка сборки
  • main.cpp - главный файл
  • timirbaevmainwindow.cpp/h/ui - главное окно
  • timirbaevdialog.cpp/h/ui - диалоговые окна
  • timirbaevadddialog.cpp/h/ui - диалог добавления
  • timirbaevautocourier.cpp/h - автокурьер
  • timirbaevcourier.cpp/h - курьер
  • timirbaevpark.cpp/h - парк транспорта
  • timirbaevtable.cpp/h - таблица данных
  • timirbaevutilities.h - утилиты
  • CMakeLists.txt - конфигурация CMake

Требования

  • Qt 6.x
  • CMake 3.16+
  • Doxygen 1.9+ (for documentation)
  • Graphviz (for diagrams)
